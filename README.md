## Installation & Setup

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```
   The site will open at `http://localhost:3000`

3. **Build for Production**
   ```bash
   npm run build
   ```

4. **Preview Production Build**
   ```bash
   npm run preview
   ```

## Design Features

### Color Palette
- Primary Green: `#1a5f3f` - Represents growth and peace
- Gold: `#d4af37` - Islamic elegance
- Sage Green: `#8b9d83` - Calm and natural
- Cream: `#f5f1e8` - Warm background
- Teal: `#2d7b7b` - Accent color

### Typography
- **Primary Font**: Poppins (clean, modern)
- **Arabic Font**: Amiri (authentic Arabic typography)

### Sections

1. **Hero** - Welcoming introduction with Bismillah
2. **About** - Importance of Islamic Madarsa education
3. **Courses** - Comprehensive program offerings
4. **Environment** - Daily schedule and facilities
5. **Impact** - Community statistics and testimonials
6. **Gallery** - Visual representation of life at Madarsa
7. **Contact** - Inquiry form and contact information
8. **Footer** - Links, social media, and donation

## Customization

### Update Content
- Edit component files in `src/components/` to modify text
- Update course offerings in `Courses.jsx`
- Change contact information in `Contact.jsx`

### Modify Colors
- Edit CSS variables in `src/index.css`
- Colors are centralized for easy theming

### Add Images
- Replace gradient placeholders in `Gallery.jsx` with actual images
- Add images to `/public/images/` folder
- Update image sources in components
